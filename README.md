# dde-calendar
Calendar for Deepin Desktop Environment, modified by Sea Monster.

### Build & Install

**Build:**
```
$ cd Deepin-Calendar-Mod-master
$ mkdir build
$ cd build
$ qmake ..
$ make
```

**Install:**
```
$ sudo make install
```
