# dde-calendar
Calendar for Deepin Desktop Environment, modified by Sea Monster.
![alt text](https://s14.postimg.org/4geulplyn/sidebyside.png?dl=1)

### Build & Install

Build:
```
$ cd Deepin-Calendar-Mod-master
$ mkdir build
$ cd build
$ qmake ..
$ make
```

3. Install:
```
$ sudo make install
```
